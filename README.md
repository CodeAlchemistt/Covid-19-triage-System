# 🦠 COVID-19 Patient Risk Triage System

A production-grade web application designed to simulate medical-grade patient risk assessment and triage for COVID-19 symptom profiles.

## 🏗️ System Architecture

This project focuses on delivering a seamless, highly available user experience utilizing a decoupled frontend/backend architecture:

- **Frontend (User Interface):** A responsive, modern clinical dashboard built strictly with HTML and **Tailwind CSS**, ensuring accessibility and clean data presentation.
- **Backend (Prediction Engine):** A **Python Flask** REST API that processes patient symptom inputs, calculates risk stratification, and serves predictions back to the UI in real time.
- **Reliability:** Features self-healing server mechanics and strict error handling to prevent browser crashes during malformed inputs or missing script tags.
